python 文件

文件操作：

打开文件：

打开文件需要用到open()函数：

open(name [, mode] [, buffering] [,encoding=None] [,errors=None,] [,newline=None] [,closefd=True] [,opener=None])

  1. 其中name表示要打开的文件，既可以是相对路径也可以是绝对路径；
  2. mode用来设置文件打开的方式；
  3. buffering是设置缓冲区策略，默认值是-1，当buffering的值为-1时，缓冲区会自动设置为4096或8192字节；当buffering的值为0时，则表示关闭缓冲区，关闭缓冲区后，数据直接写入文件中，主要用于二进制文件的写入；当buffering>0时，用来调节缓冲区的大小
  4. encoding是指用来打开文件的编码，主要用于文本文件；
  5. errors是表示编码错误时，采取的措施；
  6. newline用来设置换行模式；
  7. closefd为True时，可调用close()关闭文件，同时也会文件描述符所对应的文件；closefd为False时，不会关闭文件描述符对应的文件；
  8. opener用于打开文件的一些操作，会返回一个文件描述符

文件打开后会拥有相关属性:

file.softspace：如果用print输出后，必须跟一个空格符，则返回false。否则返回true。

关闭文件：

关闭文件用到close()函数。文件操作时常常会抛出异常，调用close函数常常放在异常处理的finally代码块中。但最好还是使用with as进行自动资源管理。

在Windows中, open方法打开的文件好像默认为GBK编码，如果你的文本文件用的utf-8格式，就会错误。

文本文件读写：

  1. read(size=-1)：从文件中读取字符串，size是最多读取的字符数，默认为-1，表示全部读取
  2. readline(size=-1)：读取到换行符或者文件尾并返回单行字符串
  3. readlines(hint=-1)：读取文件到一个字符串列表中，每一个行数据就是列表的一个元素，hint是限制读取的行数，默认值是-1，表示无限制
  4. write(s)：将字符串s写入文件，并返回写入的字符数
  5. writelines(lines)：向文件中写入一个列表，不添加分隔符，因此通常为每一行末尾提供分隔符
  6. flush()：刷新写缓冲区，数据会写入到文件中

二进制文件读写：

二进制读写方式与文本差不多，但要注意的是打开文件的方式必须是rb,wb,ab等等。

二进制文件中都是一个一个的字节数据。因此读和写，都是针对字节数据的。

文件定位：

tell()方法告诉文件内的当前位置, 即下一次的读写会发生在当前位置之后。

seek（offset [,from]）方法改变当前文件的位置。Offset变量表示要移动的字节数。From变量指定开始移动字节的参考位置。

如果from被设为0，这意味着将文件的开头作为移动字节的参考位置。

如果设为1，则使用当前的位置作为参考位置。

如果它被设为2，那么该文件的末尾将作为参考位置。

OS模块

python对文件的操作是通过io模块中的文件对象实现的。如果想用python来管理文件或目录，如删除文件，重命名等等，可以通过python的os模块实现。

  1. os.rename(source,file)：重命名文件，前面一个参数是当前的文件名，后一个是新文件名
  2. os.remove(path)：删除文件，参数是文件名
  3. os.getcwd()：显示当前工作目录名
  4. os.mkdir(path)：创建目录，在当前目录下创建新的目录们 ，参数是要创建的目录名
  5. os.chdir(path)：改变当前目录，参数是改变到的目录名
  6. os.rmdir(path)：删除目录，参数是需要删除的目录名
  7. os.walk(top)：用于通过在目录树中游走输出在目录中的文件名，向上或者向下。
  8. os.listdir(path) ：输出目录下所有的子目录 ，参数是目录名

1. os.rename(source,file)

2. os.remove(path)

如果想要删除一个文件夹，可以使用：shutil.rmtree( src) ，参数是文件夹名，需要先导入shutil库。

3. os.walk(top)

os.walk() 方法是一个简单易用的文件、目录遍历器，可以帮助我们高效的处理文件、目录方面的事情。

top – 是你所要遍历的目录的地址, 返回的是一个三元组(root,dirs,files)。

root 所指的是当前正在遍历的这个文件夹的本身的地址

dirs 是一个 list ，内容是该文件夹中所有的目录的名字(不包括子目录)

files 同样是 list , 内容是该文件夹中所有的文件(不包括子目录)

该方法没有返回值。

简单应用

OS.path 模块

对文件进行操作经常需要用到路径，而os.path模块对路径，文件，目录等管理的函数。

os.path.abspath(path)：返回path的绝对路径

os.path.nasename(path)：如果path指向文件，就返回文件名；如果path指向目录，就返回目录名

os.path.dirname(path)：返回path中的目录名

os.path.exists(path)：判断path文件是否存在

os.path.isfile(path)：如果path是文件，就返回True

os.path.isdir(path)：如果path是目录，就返回True

os.path.datetime(path)：返回最后一次访问时间，如果文件不存在或无法访问，就返回OSError

os.path.getmtime(path)：返回最后修改时间

os.path.getctime(path)：返回创造时间

os.path.getsize(path)：返回文件大小，以字节为单位
